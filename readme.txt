== Description ==

https://www.youtube.com/watch?v=Gtyqj04RqFc

This is add-on for **[Page Builder by AZEXO](https://wordpress.org/plugins/page-builder-by-azexo/)**.


= Email-template formats supported =

- **[StampReady](https://themeforest.net/tags/stampready)**


= Newsletter-plugins supported =

- **[Newsletter](https://wordpress.org/plugins/newsletter)**
- **[Email subscribers](https://wordpress.org/plugins/email-subscribers)**


== Installation ==

1. Upload the `azh_email_templates` folder to your plugins directory (e.g. `/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions
